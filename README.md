# InstaLoader
Bulk Video Reels Downloader via Username Instagram

### Perintah Termux :
    $ apt update -y && apt upgrade -y
    $ pkg install git
    $ pkg install python
    $ git clone https://github.com/dafidxcode/Insta
    $ cd InstaX
    $ python Run.py

### Update Script :
    $ rm -rf $HOME/Insta
    $ git clone https://github.com/dafidxcode/Insta
    $ cd InstaX
    $ python Run.py
